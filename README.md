# CyberShell API

Бэкенд-приложение на TypeScript для управления компьютерным клубом. Проект разработан для практики проектирования серверной архитектуры на Node.js и Express без использования сторонних ORM (база данных хранится в оперативной памяти).

## Функционал
* Управление игровыми зонами клуба (Common, VIP, Streamer).
* Ролевая модель доступа для пользователей и администраторов.
* Валидация входящих запросов в рантайме с помощью Zod.
* Управление игровыми сессиями и тарификацией на основе баланса.

## Стек технологий
* Node.js / Express
* TypeScript (strict mode)
* Zod (Runtime Validation)
* tsx / esbuild
